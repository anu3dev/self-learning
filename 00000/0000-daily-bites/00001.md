### WebClient

- Spring WebClient is part of the Spring WebFlux module and is a non-blocking and reactive web client to perform HTTP requests.
- It is also the replacement for the older RestTemplate.

```java
webClient.post()
    .uri("https://api.openai.com/v1/chat/completions")
    .header("Authorization", openApiKey)
    .bodyValue(request)
    .retrieve()
    .bodyToMono(OpenAiApiResponse.class)
    .block();
```


### annotation

- @RestController - Indicates that the class is a RESTful controller where every method returns a domain object instead of a view.
- @RequestMapping("api") - Maps HTTP requests to handler methods of MVC and REST controllers.
- @PostMapping("/chat/completions") - Specialized version of @RequestMapping for POST requests.
- @RequestBody - Binds the HTTP request body to a method parameter.

- @Entity - Specifies that the class is an entity and is mapped to a database table.


### useRef

```react
const inputRef = useRef(null);

useEffect(() => {
    inputRef.current.focus(); // Focus the input element on component mount

    inputRef..current?.scrollIntoView({ behavior: "smooth" }); // Scroll the element into view
}, []);

<input type="text" ref={inputRef} />
```
